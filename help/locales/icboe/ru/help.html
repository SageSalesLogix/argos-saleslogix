Быстрые подсказки

  • Группы доступны для предложений и заказов на продажу.
  • Кнопка "Удалить" доступна в элементах строки "Предложение" и "Заказ". Нажмите ее, чтобы удалить текущую запись из CRM. Эта функция доступна только в подробном виде элемента строки.

Поиск (только ICBOE)

Следующие поля поиска доступны, если в установленной системе Infor CRM имеется интеграция Infor CRM Back Office Extension и они дополняют ее в справке Infor CRM:

  • Клиенты

    Поля поиска

    • ID заказчика в ERP: начинается с

  • Ответственные за клиентов лица

    Поля поиска

    • Имя: начинается с

  • Бэк-офисы

    Поля поиска

    • Имя: начинается с
    • Логический ID: начинается с

  • Объект учета бэк-офиса

    Поля поиска

    • Имя: начинается с

  • Клиенты цели счета

    Поля поиска

    • Имя цели счета: начинается с

  • Клиент контакта

    Поля поиска

    • Имя контакта: начинается с

  • Счета

    Поля поиска

    • Имя клиента: начинается с
