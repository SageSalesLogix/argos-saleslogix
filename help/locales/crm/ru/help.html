        • Чтобы выводить телефонные номера в виде ссылок, выберите столбец Номер телефона, нажмите Редактировать, в раскрывающемся списке Формат выберите Номер телефона и нажмите ОК.
        Для получения дополнительной информации о редактировании групп см. справку веб-клиента или Windows-клиента.

    Автономная работа

      Если мобильный клиент работает автономно или подключение разорвано, функции позволяют получить доступ только для чтения с целью выбора записей и сущностей.
    • В разделе Недавно просмотренные отображаются все просмотренные подробные представления, благодаря чему они доступны при отсутствии подключения. Элемент «Недавно просмотренные» доступен в глобальном меню.
    • Элемент Мой портфель позволяет выбрать записи для доступа при отсутствии подключения. Чтобы добавить элементы в «Мой портфель», откройте запись в подробном представлении и щелкните значок портфеля в строке заголовка. Все соответствующие данные этой записи будут перемещены в «Портфель». Если открыть запись в виде «Мой портфель», строка заголовка имеет серый цвет, так как «Портфель» всегда находится в автономном режиме.
    • Элемент Мое расписание отображает сегодняшние встречи. «Мое расписание» доступно в глобальном меню. Все встречи из списка «Мое расписание» можно переместить в «Мой портфель». Откройте для этого список Мое расписание и щелкните значок портфеля в строке заголовка.
    • Заметки, созданные при автономной работе, будут синхронизированы при следующем входе в Infor CRM.

    Поиск

    • Существует несколько возможностей поиска:

      • С помощью глобального меню можно найти все связанные субъекты, контакты, наводки, сделки, встречи, заметки/историю и заявки. Введите дополнительный текст, чтобы уточнить критерии поиска.
      • В контекстном меню можно фильтровать результаты в списке, используя предварительно заданные ключевые слова с хэштегами.
      • Для каждой сущности результаты поиска можно отфильтровать по перечисленным ниже параметрам.

    • Субъекты

      Поля поиска

      • Имя субъекта: начинается с

    • Контакты

      Поля поиска

      • Имя: содержит
      • Фамилия: содержит
      • Имя — фамилия, имя: содержит
      • Имя субъекта: содержит

    • Наводки

      Поля поиска

      • Название компании: начинается с
      • Имя: содержит
      • Фамилия: содержит
      • Имя — фамилия, имя: содержит

    • Заявки

      Поля поиска

      • Номер: начинается с
      • Суффикс: начинается с
      • Тема: начинается с
      • Имя субъекта: начинается с

    • Встречи

      Поля поиска

      • Тема: содержит

    • Мои встречи

      Поля поиска

      • Тема: содержит

    • Мои вложения

      Поля поиска

      • Описание вложения: содержит

    • Заметки/история

      Поля поиска

      • Тема: содержит

    • Сделки

      Поля поиска

      • Имя сделки: начинается с
      • Имя субъекта: начинается с

    • Продукты (в рамках сделки)

      Поля поиска

      • Продукт: начинается с
