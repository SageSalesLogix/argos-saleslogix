  • Выберите этот значок, чтобы перейти в глобальное меню, где можно выполнить следующие действия:
    • Выполнить поиск с помощью функции SpeedSearch (в Saleslogix 8.0 или более поздней версии).
    • Использовать меню Быстрые действия, чтобы быстро добавить клиента/контакт.
    • Использовать меню Перейти к, чтобы перейти ко всем типам записей в мобильном клиенте.
    • Использовать функцию Настроить меню, чтобы изменить элементы в меню «Перейти к» и настроить порядок их отображения. Первый пункт в списке окна «Настроить меню» станет представлением по умолчанию и будет загружаться первым после входа в систему.
    • Используйте вкладку Настройки, чтобы очистить сохраненные учетные данные и хранилище, получить доступ к журналу ошибок, очистить портфель, с помощью очистки автономных параметров, и установить 12-часовой или 24-часовой формат времени в мобильном клиенте.
    • Функция «Очистить хранилище» предназначена только для устранения проблем; она позволяет удалить локально сохраненную конфигурацию.
    • Ознакомиться со справкой.

        • Чтобы выводить телефонные номера в виде ссылок, выберите столбец Номер телефона, нажмите Редактировать, в раскрывающемся списке Формат выберите Номер телефона и нажмите ОК.
        Для получения дополнительной информации о редактировании групп см. справку веб-клиента или клиента Windows.

Быстрые действия

  • Выберите в списках значок записи, чтобы открыть доступ к ярлыкам часто используемых функций. Доступные параметры зависят от типа записи. (Некоторые значки быстрых действий могут быть недоступны. Например, в разделе «Мои действия» будет недоступно действие «Отклонить», если пользователь уже подтвердил участие во встрече.) «Мои действия» и «Вложения» доступны в Saleslogix версии 8.0 и выше.
  • Используйте элемент Конфигурировать, чтобы изменить видимость и порядок элементов на панели «Быстрое действие».

Автономная работа

    Если мобильный клиент работает автономно или подключение разорвано, три функции позволяют получить доступ только для чтения с целью выбора записей и сущностей.
  • В разделе Недавно просмотренные отображаются все просмотренные подробные виды, благодаря чему они доступны при отсутствии подключения. Элемент «Недавно просмотренные» доступен в глобальном меню.
  • Элемент Мой портфель позволяет выбрать записи для доступа при отсутствии подключения. Чтобы добавить элементы в «Мой портфель», откройте запись в подробном виде и выберите значок портфеля в строке заголовка. Все соответствующие данные этой записи будут перемещены в «Портфель». Если открыть запись в виде «Мой портфель», строка заголовка имеет серый цвет, так как «Портфель» всегда находится в автономном режиме.
  • Элемент Мое расписание отображает сегодняшние действия. «Мое расписание» доступно в глобальном меню. Все действия из списка «Мое расписание» можно переместить в «Мой портфель». Откройте для этого список Мое расписание и выберите значок портфеля в строке заголовка.
    • Примечание: работа в автономном режиме не поддерживается в Windows Phone 8.

Поиск

  • Существует несколько возможностей поиска:

    • С помощью глобального меню можно найти всех связанных клиентов, контакты, заинтересованных лиц, потенциальные сделки, действия, заметки/историю и заявки. Введите дополнительный текст, чтобы уточнить критерии поиска.
    • В контекстном меню можно фильтровать результаты в списке, используя предварительно заданные ключевые слова с хэштегами.
    • Для каждой сущности результаты поиска можно отфильтровать по перечисленным ниже параметрам.

  • Клиенты

    Поля поиска

    • Имя клиента: начинается с

  • Контакты

    Поля поиска

    • Имя: начинается с
    • Фамилия: начинается с
    • NameLF: содержит
    • Имя клиента: содержит

  • Заинтересованные лица

    Поля поиска

    • Название компании: начинается с
    • Имя: начинается с
    • Фамилия: начинается с
    • NameLF: содержит

  • Заявки

    Поля поиска

    • Номер: начинается с
    • Суффикс: начинается с
    • Тема: начинается с
    • Имя клиента: начинается с

  • Действия

    Поля поиска

    • Тема: содержит

  • Мои действия

    Поля поиска

    • Тема: содержит

  • Мои вложения

    Поля поиска

    • Описание вложения: содержит

  • Заметки/история

    Поля поиска

    • Тема: содержит

  • Потенциальные сделки

    Поля поиска

    • Имя потенциальной сделки: начинается с
    • Имя клиента: начинается с

  • Продукты (в рамках потенциальной сделки)

    Поля поиска

    • Продукт: начинается с
