Быстрые подсказки

  • Коснитесь этого значка, чтобы перейти в глобальное меню, где вы сможете:
    • Выполнить поиск с помощью функции SpeedSearch (в Saleslogix 8.0 или более поздней версии).
    • Использовать Быстрые действия для удобного добавления учетной записи или контакта.
    • Воспользоваться функцией Перейти для доступа ко всем типам записей в Mobile Client.
    • Настроить меню, изменив объекты в меню перехода и выбрав порядок их отображения. Первый объект в списке "Настройка меню" становится представлением по умолчанию и загружается первым при входе в систему.
    • Использовать Настройки для удаления сохраненных учетных данных, очистки хранилища и журналов ошибок доступа.
    • Действие очистки хранилища в настройках используется исключительно для устранения проблем и позволяет очистить сохраненные локально настройки.
    • Ознакомиться со справкой.
    • Выйти из системы.
    • Элемент Подключение показывает, подключен ли Mobile Client к сети (синяя строка заголовка в записях) или нет (серая строка заголовка).
  • Нажмите этот значок, чтобы перейти в контекстное меню (справа). Доступные параметры зависят от типа записи.
    • Доступные хэштеги
    • Ключевым показателям эффективности.
    • Разверните Группы, нажмите Настроить и выберите группы, которые будут отображаться в списках. Данная функция доступна в списках клиентов, контактов, заинтересованных лиц, возможностей и заявок.
    • Выбрать представление. Эта функция доступна для списков клиентов, контактов, заинтересованных лиц, возможностей и заявок. Выбранное представление задается по умолчанию при последующем возврате к этому типу записи.
      • Режим сводного представления показывает четыре поля в одном столбце.
      • В подробном представлении отображаются поля в 1–3 столбцах в зависимости от ширины экрана устройства.
        Примечание. В версиях CRM Web или Windows Client есть возможность изменить отображение некоторых полей в сводном и подробном мобильных представлениях.
        • В списке клиентов, контактов, заинтересованных лиц, возможностей или заявок Web Client щелкните правой кнопкой вкладку группы, выберите Изменить, а затем выберите вкладку Представление.
        • Используйте перетаскивание, чтобы изменить расположение столбцов.
        • Чтобы выводить денежные суммы зеленым цветом, выберите столбец Потенциал, нажмите Изменить, в раскрывающемся списке Формат выберите значение Валютаи нажмите ОК.
        • Чтобы выводить телефонные номера в виде ссылок, выберите столбец Номер телефона, нажмите Изменить, в раскрывающемся списке Формат выберите Номер телефона и нажмите ОК.
        Узнать больше о редактировании групп вы можете из справки Web или Windows Client.

Быстрые действия

  • Выберите в списках значок записи, чтобы открыть доступ к ярлыкам часто используемых функций. Доступные параметры зависят от типа записи. (Некоторые значки быстрых действий могут быть недоступны. Например, в разделе "Мои действия" будет недоступно действие "Отклонить", если пользователь уже подтвердил участие во встрече.) "Мои действия" и "Вложения" доступны в Saleslogix версии 8.0 и выше.
  • Используйте элемент Конфигурировать, чтобы изменить видимость и порядок элементов на панели "Быстрое действие".

Индикаторы действия

  • Просрочено
  • Оповещение
  • Последнее изменение
  • Приоритет
  • Повторение

Автономная работа

    Если Mobile Client не подключен или подключение разорвано, эти функции позволяют получить доступ только для чтения для выбора записей и объектов.
  • В разделе Недавно просмотренные отображаются все просмотренные подробные виды, благодаря чему они доступны при отсутствии подключения. Элемент "Недавно просмотренные" доступен в глобальном меню.
  • Элемент Мой портфель позволяет выбрать записи для доступа при отсутствии подключения. Чтобы добавить элементы в "Мой портфель", откройте запись в подробном виде и выберите значок портфеля lв строке заголовка. Все соответствующие данные этой записи будут перемещены в "Портфель". Если открыть запись в виде "Мой портфель", строка заголовка имеет серый цвет, так как "Портфель" всегда находится в автономном режиме.
  • Элемент Мое расписание отображает сегодняшние действия. "Мое расписание" доступно в глобальном меню. Все действия из списка "Мое расписание" можно переместить в "Мой портфель". Откройте для этого список Мое расписание и выберите значок портфеля в строке заголовка.

Поиск

  • Существует несколько возможностей поиска:

    • В главном меню доступен поиск всех связанных учетных записей, контактов, потенциальных и возможных клиентов, действий, заметок/истории и заявок. Чтобы повысить точность поиска, вводите более длинные запросы.
    • В контекстном меню доступна фильтрация списков результатов с помощью предустановленных ключевых хэштегов.
    • Результаты поиска можно отфильтровать по следующим параметрам:

  • Клиенты

    Поля поиска

    • Имя клиента: начинается с

  • Контакты

    Поля поиска

    • Имя: начинается с
    • Фамилия: начинается с
    • NameLF : содержит

  • Заинтересованные лица

    Поля поиска

    • Название компании: начинается с
    • Имя: начинается с
    • Фамилия: начинается с
    • NameLF : содержит

  • Заявки

    Поля поиска

    • Номер: начинается с
    • Индекс: начинается с
    • Тема: начинается с
    • Имя клиента: начинается с

  • Действия

    Поля поиска

    • Тема: содержит

  • Мои действия

    Поля поиска

    • Тема: содержит

  • Мои вложения

    Поля поиска

    • Описание вложения: содержит

  • Заметки/история

    Поля поиска

    • Тема: содержит

  • Возможности

    Поля поиска

    • Имя возможности: начинается с
    • Имя клиента: начинается с

  • Продукты (в рамках возможности)

    Поля поиска

    • Продукт: начинается с
