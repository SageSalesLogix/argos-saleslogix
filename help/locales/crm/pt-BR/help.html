Dicas Rápidas

  • Clique neste ícone para ir ao menu Global, onde você pode:
    • Pesquisar usado SpeedSearch
    • Usar Ações Rápidas para adicionar rapidamente uma Conta/Contato.
    • Usar Ir para para acessar todos os tipos de registro no Cliente Móvel.
    • Expanda Outros para acessar os recursos a seguir:
      • Usar Configurar Menu para alterar os itens no menu Ir para e especificar a ordem em que eles serão exibidos. O primeiro item na lista na tela Configurar Menu se tornará a sua exibição padrão e será carregado primeiro após você se conectar.
      • Usar Definições para limpar as credenciais salvas, limpar o armazenamento, acessar os logs de erro, esvaziar Minha Maleta por meio da limpeza das Opções Off-line, definir o Cliente móvel para usar o relógio de 12 horas em vez do relógio de 24 horas e definir um novo idioma para o Mobile Client.
      • Localizar a Ajuda.
      • Encerrar Sessão.
      • Conexão mostra se o seu Cliente móvel está Online (a barra de cabeçalho nos registros será exibida em azul) ou Offline (a barra de cabeçalho será exibida em cinza).
  • Clique neste ícone para acessar o menu Contextual (à direita) e as Definições de Exibição. As opções disponíveis variam de acordo com o tipo de registro.
    • Consulte Hashtags Disponíveis
    • Visualizar KPIs (Indicadores-Chave de Desempenho)
    • Expanda Grupos, Configurar para selecionar os grupos a serem exibidos nas visualizações em lista. Isso está disponível nas visualizações em listas Conta, Contato, Lead e Oportunidade e Tíquete.
    • Faça uma seleção em Layouts, disponível nas exibições Conta, Contato, Lead, Oportunidade e Lista de Tíquetes. O layout selecionado se tornará o padrão para quando você retornar a esse tipo de registro posteriormente.
      • O layout de resumo é exibido como 4 campos em 1 coluna. Esse é o padrão.
      • O layout de detalhes mostra como 9 campos em 1 coluna.
        Nota: se você tiver acesso ao Cliente Infor CRM Web ou Windows, você poderá modificar a exibição de alguns campos nesses layouts Resumidos e Detalhados para dispositivos móveis.
        • Na Exibição em Lista Conta de Cliente Web, Contato, Lead, Oportunidade ou Tíquete, clique com o botão direito do mouse em uma de grupo, clique em Editar e, depois, clique na guia Layout:
        • Arraste e solte as colunas para alterar sua ordem.
        • Exibir campos de moeda em texto verde: selecione a coluna Potencial, clique em Editar, expanda o menu suspenso do campo Tipo de Formato e selecione Moeda, clique em OK.
        • Exibir números de telefone como um link clicável: selecione Número do Telefone, clique em Editar, expanda o menu suspenso Tipo de Formato e selecione Número do Telefone, clique em OK.
        Consulte a Ajuda do Cliente Web ou Windows para obter mais informações sobre a edição de grupos.
  • Este ícone indica que você pode expandir e recolher painéis para exibir/ocultar detalhes adicionais.
  • Esse ícone indica que você pode adicionar um novo registro ao Infor CRM. Ele está disponíveis em todas as visualizações em lista.
  • Quando o botão Atualizar estiver disponível, clique nele para atualizar imediatamente as informações. Selecione um registro para limitar a atualização para esse registro. Quando nenhum registro for selecionado, os dados para todo o grupo serão atualizados
  • O texto azul em visualizações detalhadas significa uma ação de link (como navegar para uma URL ou ir para uma Conta).
  • Nota: se você estiver usando um iPhone ou iPad e acessar o Infor CRM Mobile a partir de um atalho na tela de início, o link do Mapa será aberto em uma nova exibição e o link da Web estará desabilitado.

Ações Rápidas

  • Nas visualizações em lista, toque no ícone do Ellipsis para ver e acessar atalhos aos recursos usados com frequência. As opções disponíveis variam dependendo do tipo de registro. (Alguns ícones de ação rápida podem não estar disponíveis. Por exemplo, em Minha Programação, a ação Recusar estará indisponível se você já aceitou a reunião).
  • Use Outros >Configurar Menu para alterar a visibilidade e a ordem dos itens no painel Ação Rápida.

Trabalhando Offline

    Os recursos permitem o acesso somente leitura para selecionar registros e entidades quando o seu Cliente Móvel está desconectado ou perde a conectividade.
  • Recentemente Visualizado mostra as visualizações detalhadas visitadas e disponíveis em caso de perda de conectividade. Recentemente Visualizado está disponível no Menu Global.
  • Minha Maleta permite que você selecione registros para ficarem disponível quando estiver offline. Para adicionar itens a Minha Maleta, na exibição detalhada, abra um registro e, depois, toque no ícone da maleta localizado na barra do cabeçalho. Todos os dados relacionados com esse registro também serão movidos para a Maleta. Quando você abrir um registro na exibição Minha Maleta, a barra do cabeçalho estará sempre cinza porque Minha Maleta está sempre no modo off-line.
  • Minha Programação mostra as atividades do dia. Minha Programação está disponível no menu Global. Você pode mover todas as atividades na exibição de lista Minha Programação para Minha Maleta. Para fazer isso, abra a exibição em lista Minha Programação e toque no ícone da maleta localizado na barra do cabeçalho.
  • Notas criadas durante a sincronização offline na próxima vez que você se conectar ao Infor CRM.

Pesquisa

  • Existem várias maneiras de pesquisar:

    • No menu Global, localize todas as contas, contatos, leads, oportunidades, atividades, notas/histórico e tíquetes relacionados. Insira mais texto para melhorar a precisão da sua pesquisa.
    • No menu Contextual, filtre os resultados da exibição em lista usando palavras-chave com hashtag pré-definidas.
    • Para cada entidade, filtre os resultados da exibição em lista a partir da barra de pesquisa usando os campos especificados abaixo.
