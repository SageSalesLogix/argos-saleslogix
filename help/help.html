Quick Tips

  • The configure button on the Home screen allows you to choose the types of records to show in the "Go To" section and specify the order in which they appear.
  • This icon indicates that you can expand and collapse panels to show/hide additional details.
  • This icon indicates that you can add a new record to SalesLogix. It is available on all list views.
  • Underlined text in detail views signifies a link action (such as browsing to a URL or jumping to an Account).
  • The "Clear Storage" option under Settings is strictly a trouble-shooting device to clear locally stored configuration.

Quick Actions

  • Following phone call or email actions, a view enabling you to type notes and save to history displays.
  • While in the list view, click the record icon to see and access short-cuts to frequently-used features. (Some quick action icons may be unavailable. For example, in My Activities, the Decline action will be unavailable if you have already accepted the meeting.)
  • The "View address" action takes you to the native map application on supported devices.

Search

    • The Search bar provides two ways to search, depending on how it is accessed:
    • From the Home screen, find all related accounts, contacts, leads, opportunities, activities, notes\history, and tickets. Enter more text to improve the accuracy of your search.
    • While in a record (Activities, Accounts, and so on) filter list view results based on the fields specified below. Some views allow you to enter hash tag keywords for quick filtering.
